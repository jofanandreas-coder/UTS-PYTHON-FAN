## Studi Kasus  
Program ini digunakan untuk menilai hasil performa karyawan berdasarkan tiga aspek penilaian: Kedisiplinan, Tanggung Jawab, dan Kerja Sama.  
Setiap aspek dinilai dengan rentang 1–10, kemudian dihitung rata-ratanya.  
Hasil akhir dikategorikan menjadi:  
- Excellent → jika rata-rata > 8  
- Good → jika rata-rata antara 6–8  
- Need Improvement → jika rata-rata < 6  

## Fitur Program  
1. Menginput nilai dari tiga aspek penilaian  
2. Menghitung rata-rata nilai performa  
3. Menentukan kategori hasil penilaian secara otomatis
